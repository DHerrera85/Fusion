# Fusión Preview

## Descripción
Este proyecto es una página estática de demostración para **Fusión**, una plataforma de series y proyectos audiovisuales. Incluye:

- **Galería principal (“Featured Collection”)** de imágenes cuadradas (760×760 px) en un carrusel Flickity de 3 por vista, con título superpuesto.
- **Galerías verticales** (280×420 px) que muestran 4 ítems por vista, con título, año y logo de canal.
- **Galerías horizontales** (320×180 px) que muestran 3 ítems por vista, con la misma estructura de datos.
- **Interacción táctil** y flechas de navegación para deslizar en escritorio y dispositivos móviles.
- **Diseño responsivo**, tipografía Audiowide y paleta en tonos celeste claro.

## Tecnologías
- **HTML5** y **CSS3** (incluyendo Google Fonts: Source Sans Pro y Audiowide)
- **JavaScript** puro
- **[Flickity](https://flickity.metafizzy.co/)** para los carruseles
- **Netlify** para el despliegue continuo
